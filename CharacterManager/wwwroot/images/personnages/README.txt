Instructions pour ajouter des images de personnages
===================================================

1. Placez vos fichiers PNG dans ce dossier
2. Nommez les fichiers en minuscules selon le nom du personnage
   Exemple : regina.png, ravenna.png, belle.png

3. Format recommandé :
   - Format PNG avec transparence
   - Résolution minimale : 300x400 pixels
   - Résolution recommandée : 600x800 pixels

4. Image par défaut :
   - Créez un fichier "default.png" pour les personnages sans image

5. Conventions de nommage :
   - Utilisez uniquement des lettres minuscules
   - Pas d'espaces (utilisez des tirets si nécessaire)
   - Exemple : BELLE -> belle.png
            VICTORIA -> victoria.png
